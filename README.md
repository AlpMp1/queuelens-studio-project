# QueueLens Studio

QueueLens Studio 是一个 AI 消息队列堆积诊断与消费者修复工作台，面向事件驱动系统。

## 项目能力
- 读取 Kafka/RabbitMQ/SQS 指标、消费者源码、DLQ 消息样本、重试配置、trace、日志和部署 diff
- 定位消费变慢、重复投递、幂等缺失、重试风暴和死信堆积问题
- 生成 bounded retry、poison quarantine、幂等保护、回放测试和运行手册
- 输出 PR 级修复报告、验证矩阵和后续预防任务

## 部署方式
这是一个静态 HTML demo，可以直接部署到 GitHub Pages、Netlify、Vercel 或任意静态托管平台。
